
# Getting Started
* Ex1: List the contents of the root directory.
```
ls /
```
* Ex2: Change directories to your default home directory.
```
cd ~
```
* Ex3:  Verify your current working directory is `/home/theia`
```
pwd
```
* Ex4: Use tab completion to change directories to `/bin`
```
/b and use the Tab button twice as there are 2 directories in / have names starting with a b
```

* Ex5: Use your terminal's command history to change directories back to your home directory.
```
Use up arrow until cd ~ found
```
